Yaml configuration files syntax.

## Installation
```shell
composer require mvccore/ext-config-yaml
```

## Usage
Add this to `Bootstrap.php` or to very application beginning:
```php
\MvcCore\Ext\Auth::GetInstance()->SetConfigClass('MvcCore\Ext\Config\Yaml');
```
